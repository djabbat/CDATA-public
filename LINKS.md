# LINKS.md — CDATA: Ecosystem Connections

## Internal Ecosystem Links

### CDATA ↔ Ze (Ze Vector Theory)
- **Direction:** CDATA simulation output → Ze-analysis
- **Use case:** Stem cell division stream (T=division, S=quiescence) → Ze-velocity v as aging biomarker
- **Planned article:** "CDATA + Ze: Centriolar Damage and Ze-State of Stem Cells" (NEEDTOWRITE.md)
- **Integration:** ZIO-index + χ_Ze + mito_shield → single aggregate aging index

### CDATA ↔ FCLC (Federated Clinical Learning Cooperative)
- **Direction:** CDATA synthetic datasets → FCLC orchestrator validation set
- **Use case:** CDATA aging biomarker simulations serve as held-out validation at orchestrator level
- **Benefit:** Avoids need for real patient data at orchestrator; enables Shapley scoring without privacy risk

### CDATA ↔ AIM (AI-assisted Integrative Medicine)
- **Direction:** CDATA frailty predictions → AIM patient history
- **Use case:** frailty_index and epigenetic_age as prognostic fields in AIM's Bayesian network
- **Status:** Planned integration

### CDATA ↔ HealthWearable
- **Direction:** HealthWearable RR/HRV data → CDATA validation
- **Use case:** Real-time patient HRV as proxy for stem cell Ze-velocity; cross-validation with CDATA predictions
